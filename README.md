
# Nostalgia Design Mint — Portfólio

Descrição
- Projeto de portfólio com interface inspirada no Linux Mint. Contém janelas, barra de tarefas e visual de aplicativos (Sobre, Projetos, Spotify, Contato).

Tecnologias
- React
- TypeScript
- Vite
- Tailwind CSS
- Radix UI
- Lucide Icons

Como rodar (desenvolvimento)
1. Instalar dependências:

```bash
cd "client"
npm install
```

2. Iniciar servidor de desenvolvimento:

```bash
npm run dev:client
```

Build (produção)

```bash
cd "client"
npm run build
```
